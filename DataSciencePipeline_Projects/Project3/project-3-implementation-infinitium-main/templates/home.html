About our project

The dashboard provides a visualization of our data for keywords relating abortion, politics and elections which has been collected from Twitter, Reddit and Youtube. The data can be visualized over two options:

  1. Data by Date : The program takes in a date range as an input and plots a graph depicting the data

  2. Popular Hashtags : The program takes in a keyword as an input and retrieves the top five, most popular hashtags related to the keyword

Sentiment Analysis Question

How conclusive is the sentiment of people on social media towards abortion rights, when it comes to the elections?

  • From the data collected using the various forms of social media, we found that there are a lot of records of data condemning Republicans and their views on abortion. We believe that people who support abortion will vote for Democratic candidates. However, it is not conclusive if abortion is THE factor in determining who to vote for.
